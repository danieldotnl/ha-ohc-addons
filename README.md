# Our Home Connected - Home Assistant Add-ons

This repository contains add-ons for Home Assistant.

## Add-ons

### [Our Home Connected](./ourhomeconnected)

Version control for your Home Assistant automations and scripts with GitHub integration.

## Installation

1. Navigate to the "Add-on Store" in your Home Assistant instance
2. Add this repository URL: https://github.com/danieldotnl/ha-ohc-addons
3. Install the ourhomeconnected add-on
